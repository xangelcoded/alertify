System overview

Built to understand crisis signals at scale.

Alertify ingests public disaster-related posts, applies multilingual NLP, and surfaces high-risk events on a single operational view for Lipa City’s emergency teams.

Multilingual AI engine
Uses a multilingual BERT model tuned for disaster-related content, understanding English, Tagalog, and Taglish posts common in local social media.
Urgency & hazard ranking
Scores and ranks each report based on urgency and disaster type so operators can see critical alerts for flooded roads, trapped residents, or fires.
Map + notifier dashboard
Combines a geospatial incident map with a live alert feed, optimized for LGU command centers and multi-screen operations.
